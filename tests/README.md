# Tests

Dieser Ordner sammelt spaeter uebergreifende Testartefakte.

Geplante Testarten:

- Unit Tests fuer Rule Engine und Simulation Engine.
- Integrationstests fuer Planungsszenarien.
- Regressionstests gegen dokumentierte Excel-Beispiele.
- E2E-Tests fuer kritische UI-Workflows.

In Sprint 1 werden noch keine fachlichen Tests implementiert.
